## Ana
La checklist pour ajouter un nouveau jeu est :

1. Créer [jeu]-engine.cjs dans server/games/
2. Importer l'engine dans games-routes.cjs
3. Ajouter les routes dans games-routes.cjs
4. Ajouter à la liste /games/list dans games-routes.cjs
5. Ajouter à GAMES_LIST dans GamesPage.jsx

Il est important de suivre ces étapes pour s'assurer que le nouveau jeu est correctement intégré au système.
