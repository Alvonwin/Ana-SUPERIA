Pour lancer la commande "node --version" en arrière-plan, vous pouvez utiliser le symbole "&" à la fin de la commande dans un système d'exploitation Unix-like (comme Linux ou macOS). Voici comment faire :

```bash
node --version &
```

Cette commande exécutera `node --version` en arrière-plan, ce qui signifie que le processus sera lancé mais que le contrôle du terminal vous sera rendu immédiatement, vous permettant d'exécuter d'autres commandes sans attendre la fin de l'exécution de la précédente.

Si vous utilisez Windows, vous pouvez utiliser la commande `start` pour démarrer un processus en arrière-plan :

```bash
start node --version
```

Cela ouvrira une nouvelle fenêtre de commande et exécutera la commande `node --version` dans cette fenêtre, vous laissant libre d'utiliser la fenêtre de commande originale.

Sachez que la commande `node --version` affiche simplement la version de Node.js installée sur votre système, elle ne nécessite généralement pas d'être exécutée en arrière-plan car elle se termine rapidement.
