Il semble que tu veux que je te montre les commandes Git pour afficher les différences entre les répertoires. Voici quelques commandes qui pourraient t'aider :

1. `git diff` : Affiche les différences entre les fichiers du répertoire local et le répertoire distant.
2. `git diff --staged` : Affiche les différences entre les fichiers du répertoire local et le répertoire distant, en incluant les fichiers qui sont dans la zone de préparation.
3. `git diff --cached` : Affiche les différences entre les fichiers du répertoire local et le répertoire distant, en incluant les fichiers qui sont dans la zone de préparation et les fichiers qui ont été modifiés localement.
4. `git diff --no-commit` : Affiche les différences entre les fichiers du répertoire local et le répertoire distant, sans inclure les fichiers qui ont été modifiés localement.
5. `git diff --stat` : Affiche les différences entre les fichiers du répertoire local et le répertoire distant, en incluant les statistiques sur les modifications.
