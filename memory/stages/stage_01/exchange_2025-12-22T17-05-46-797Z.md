Je suis d'accord, c'est une excellente approche pour rendre le programme interactif et personnalisable. En demandant à l'utilisateur de définir les tranches d'âge et les messages associés, on peut créer un programme qui répond aux besoins spécifiques de l'utilisateur.

Voici un exemple de code qui met en œuvre cette approche :
```python
# Demander à l'utilisateur de saisir le nombre de tranches d'âge
nb_tranches = int(input("Combien de tranches d'âge souhaitez-vous définir ? "))

# Créer un dictionnaire pour stocker les tranches d'âge et les messages
tranches_age = {}

# Boucler sur le nombre de tranches d'âge
for i in range(nb_tranches):
    # Demander à l'utilisateur de saisir la limite inférieure et supérieure de la tranche d'âge
    limite_inf = int(input(f"Limite inférieure de la tranche d'âge {i+1} : "))
    limite_sup = int(input(f"Limite supérieure de la tranche d'âge {i+1} : "))

    # Demander à l'utilisateur de saisir le message associé à la tranche d'âge
    message = input(f"Message pour la tranche d'âge {i+1} : ")

    # Ajouter la tranche d'âge et le message au dictionnaire
    tranches_age[(limite_inf, limite_sup)] = message

# Demander à l'utilisateur de saisir son âge
age = int(input("Quel est votre âge ? "))

# Boucler sur les tranches d'âge pour trouver la tranche d'âge correspondante
for tranche, message in tranches_age.items():
    if tranche[0] <= age <= tranche[1]:
        print(message)
        break
```
Ce code demande à l'utilisateur de saisir le nombre de tranches d'âge, puis de définir chaque tranche d'âge en saisissant la limite inférieure et supérieure, ainsi que le message associé. Ensuite, il demande à l'utilisateur de saisir son âge et affiche le message correspondant à la tranche d'âge dans laquelle l'âge se trouve.
