C'est une excellente idée ! En ajoutant un bloc `except ValueError` pour gérer les erreurs de saisie non numérique et en affinant les conditions pour distinguer les différentes tranches d'âge, on peut rendre le programme plus complet et interactif.

Voici un exemple de code qui montre comment on peut implémenter cela :
```python
# Demander à l'utilisateur son nom et son âge
nom = input("Quel est votre nom ? ")

# Demander à l'utilisateur son âge et convertir en nombre entier
while True:
    try:
        age = int(input("Quel est votre âge ? "))
        break
    except ValueError:
        print("L'âge doit être un nombre entier.")

# Répéter une petite phrase plusieurs fois en boucle
for i in range(5):
    print("Bonjour, " + nom + "!")

# Utiliser des conditions pour afficher un message différent selon l'âge donnée
if age < 13:
    print("Vous êtes un enfant.")
elif age < 18:
    print("Vous êtes un adolescent.")
elif age < 26:
    print("Vous êtes un jeune adulte.")
elif age < 65:
    print("Vous êtes un adulte.")
else:
    print("Vous êtes un senior.")
```
Ce code ajoute un bloc `except ValueError` pour gérer les erreurs de saisie non numérique et affine les conditions pour distinguer les différentes tranches d'âge. On peut ainsi personnaliser les messages pour rendre le programme plus interactif et complet.
