Il semble que vous cherchiez à modifier la variable d'environnement PATH. La variable PATH est utilisée pour spécifier les répertoires où Windows recherche les exécutables. 

Pour modifier la variable PATH, vous pouvez suivre ces étapes :

1.  Ouvrez l'invite de commande en tant qu'administrateur. 

2.  Tapez la commande suivante pour afficher la variable PATH actuelle :

`echo %PATH%`

3.  Modifiez la variable PATH en ajoutant ou en supprimant les répertoires souhaités.

4.  Tapez la commande suivante pour mettre à jour la variable PATH :

`set PATH=new_path`

Remplacez `new_path` par le chemin d'accès au répertoire que vous souhaitez ajouter.

5.  Fermez l'invite de commande.

Notez que les modifications apportées à la variable PATH ne sont pas permanentes et seront perdues à la prochaine réinitialisation du système.
