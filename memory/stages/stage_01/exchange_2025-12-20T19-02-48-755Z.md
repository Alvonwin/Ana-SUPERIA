9. Qu'est-ce qu'un type de données composite ?
Un type de données composite est un type de données qui combine plusieurs types de données de base, comme un tableau ou un objet.

10. Quelle est la différence entre un tableau et un objet ?
Un tableau est un type de données qui représente une collection de valeurs de même type, stockées dans un ordre spécifique, tandis qu'un objet est un type de données qui représente une collection de paires clé-valeur.
