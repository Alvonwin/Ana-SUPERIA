## Ana
 Pour lancer une commande en arrière-plan (background) sur Windows, vous pouvez utiliser le symbole `&` ou placer la commande entre guillemets. Voici comment vous pouvez exécuter `ping localhost -n 2` en arrière-plan :

```shell
ping localhost -n 2 &
```

Cette commande lancera une boucle de ping sur votre machine locale pendant 2 secondes et retournera immédiatement à l'invite de commande. Si vous souhaitez voir le résultat du ping, vous pouvez simplement exécuter la commande sans le symbole `&`.

Pour Linux ou Mac :
```shell
ping localhost -c 2 &
```

Cette commande lancera une boucle de ping sur votre machine locale pendant 2 secondes et retournera immédiatement à l'invite de commande. Si vous souhaitez voir le résultat du ping, vous pouvez simplement exécuter la commande sans le symbole `&`.

Assurez-vous que votre machine locale est correctement configurée pour permettre les pings depuis l'extérieur.
