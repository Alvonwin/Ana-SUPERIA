Voici un exemple de fonction Python qui trie une liste par ordre décroissant :

```python
def trier_liste_decroissante(liste):
    return sorted(liste, reverse=True)
```

Cette fonction prend une liste en entrée et utilise la fonction `sorted()` pour la trier. Le paramètre `reverse=True` indique que la liste doit être triée dans l'ordre décroissant.

Voici comment vous pouvez utiliser cette fonction :

```python
ma_liste = [5, 2, 9, 1, 5, 6]
liste_triee = trier_liste_decroissante(ma_liste)
print(liste_triee)  # Output: [9, 6, 5, 5, 2, 1]
```

Cette fonction est simple et efficace pour trier une liste en Python.
