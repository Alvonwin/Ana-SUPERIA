Pour lancer la commande "node --version" en arrière-plan, vous pouvez utiliser les symboles "&" à la fin de la commande sous Linux ou macOS, ou utiliser la commande "start" sous Windows.

**Sous Linux ou macOS :**
```bash
node --version &
```
Cette commande exécutera "node --version" en arrière-plan et affichera la version de Node.js installée sur votre système.

**Sous Windows (via l'invite de commande) :**
```bash
start node --version
```
Cette commande ouvrira une nouvelle fenêtre de commande qui exécutera "node --version" et affichera la version de Node.js installée sur votre système.

**Remarque :** Si vous utilisez un gestionnaire de fenêtres ou un environnement de développement intégré (IDE), les méthodes pour lancer des commandes en arrière-plan peuvent varier.
