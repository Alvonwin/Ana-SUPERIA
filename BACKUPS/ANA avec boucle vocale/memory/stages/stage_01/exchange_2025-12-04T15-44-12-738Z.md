1. **Installer Python** : Assurez-vous que Python est installé sur votre système. Vous pouvez télécharger la dernière version de Python depuis le site officiel : https://www.python.org/downloads/

2. **Ouvrir une invite de commandes administrateur** :
   - Recherchez "invite de commandes" dans la barre de recherche.
   - Cliquez droit sur "Invite de commandes" et sélectionnez "Exécuter en tant qu'administrateur".

3. **Installer Jupyter Notebook via pip** :
   - Dans l'invite de commandes, entrez la commande suivante et appuyez sur Entrée :
     ```sh
     pip install notebook
     ```

4. **Lancer Jupyter Notebook** :
   - Une fois l'installation terminée, vous pouvez démarrer Jupyter Notebook en tapant la commande suivante dans l'invite de commandes :
     ```sh
     jupyter notebook
     ```
   - Cela ouvrira Jupyter Notebook dans votre navigateur web par défaut.
