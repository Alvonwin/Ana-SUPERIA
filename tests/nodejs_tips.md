# Meilleures pratiques Node.js

## 1. La Ferme du Web
[La Ferme du Web](https://www.lafermeduweb.net/veille/une-liste-des-bonnes-pratiques-a-suivre-en-node-js) propose une liste de bonnes pratiques en Node.js basée sur des articles pertinents.

## 2. GitHub - goldbergyoni/nodebestpractices
[Github goldbergyoni/nodebestpractices](https://github.com/goldbergyoni/nodebestpractices) offre une liste complète de meilleures pratiques pour Node.js, couvrant des aspects tels que l'utilisation de versions LTS, la gestion des environnements, la sécurité et le déploiement.

## 3. Meilleures pratiques en termes de performances pour Express en production
[L'équipe d'Express](https://expressjs.com/fr/advanced/best-practice-performance.html) recommande plusieurs meilleures pratiques pour améliorer les performances des applications Express, notamment la mise en cache des modèles et des fichiers CSS générés.
